FastAPI + Postgres example

Postgres Playground API

This project is the reference service for the hybrid architecture: FastAPI app code in its own folder, a dedicated Postgres schema, Docker deployment on the shared platform, and a health endpoint for Traefik or direct host routing.

Why it exists

An example service that proves the repo can host a non-trivial backend project with its own tooling, schema, and deploy path.

The main site stays easy to publish, while project services can live on shared host infrastructure with Traefik, cloudflared, and one Postgres instance carved into project schemas.
