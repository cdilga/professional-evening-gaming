Bounce

Bounce is the lightweight front-end game slot in the system: easy to ship, easy to link from the main site, and a good example of a project that mostly wants static assets and a tiny deploy surface.

Why it exists

The original browser game experiment stays visible as the playful north star: a small, tactile prototype with room to get weirder.

The main site stays easy to publish, while project services can live on shared host infrastructure with Traefik, cloudflared, and one Postgres instance carved into project schemas.
