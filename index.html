Rediscovered vision

The original direction was not just a cute static page. It was a calm front door for a growing collection of experiments, backed by one repeatable platform: shared ingress, per-project boundaries, and a cheap path from local tinkering to something more durable.

  • One main site with clear project submenus
  • One repo that can host mixed project tooling without collapsing
  • One Postgres instance with project-specific schemas
  • One example service proving the pattern end to end

Hybrid PPaaS

The main site stays easy to publish, while project services can live on shared host infrastructure with Traefik, cloudflared, and one Postgres instance carved into project schemas.
